# 🎓 AI-Powered College Enquiry Chatbot

## 📌 Project Overview
The **AI-Powered College Enquiry Chatbot** is a smart web-based application built using **Base44** that helps students get instant and accurate information about a college.  

It acts as a virtual assistant that answers queries related to:
- Admissions process
- Courses and specializations
- Fee structure
- Hostel facilities
- Scholarships
- Placement statistics
- Academic calendar
- Contact information

The system reduces the need for manual enquiry and improves student experience by providing **24/7 automated responses**.
